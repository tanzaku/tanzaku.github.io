ThinkPad_T14_Gen1(AMD)にUbuntu20.10をインストールした

 December 18, 2020に投稿  |  1 分間  |  108 言葉  |  tanzaku
  1. 回復ドライブの作成
  2. Bitlocker（デバイスの暗号化）をオフに
  3. 高速スタートアップを無効化
  4. UbuntuのライブUSBを差し込み、シャットダウンして起動し、起動中にF12連打
  5. bootメニューからUSBを選んでインストール（今回はデュアルブートにした）
  6. ThinkPad T14 Gen1 (AMD)にUbuntu 20.10をインストールした際のメモ
    1. BIOSの電源スリープの設定を変更する
    2. Canonicalパートナーのリポジトリを有効にする
    3. 音は普通に鳴ったのでそのまま
  7. アプリのインストールなど
    1. mozc追加(Ubuntu 20.04で日本語入力 (iBus – Mozc, Fcitx – Mozc)│きなこあんこん)
    2. chromeインストール(UbuntuにChromeをインストールする - Qiita)
    3. g++, curl, git, wget, java(default-jre default-jdk), rustup, g++, curl, git, wget, python, golang, docker, docker-compose, vscode, typora
    4. 指紋認証ログイン有効化(Settings -> Users -> Fingerprint Login)
    5. chromeにlastpassインストール
